Bienvenue sur ma page web

Utilisez les touches gauche et droite pour naviguer

Dans cet exemple, la classe "container" a une largeur de 100% et une hauteur de 200 pixels. La propriété "overflow" est définie sur "auto", ce qui signifie que la barre de défilement apparaîtra uniquement si le contenu dépasse la hauteur de l'élément. La règle media query est utilisée pour définir la largeur de l'élément "container" à 50% sur les écrans plus grands (à partir de 768 pixels de largeur). Grâce à cette approche, l'élément "container" aura toujours une barre de défilement disponible, quelle que soit la taille de l'écran de l'appareil utilisé pour afficher la page web.
